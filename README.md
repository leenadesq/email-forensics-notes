## Purpose

This repository is a working knowledge base for email forensics and mailbox migration work. Each file focuses on what IT admins, e-discovery analysts, and forensic specialists actually do in practice, with references checked against format specifications and real-world tools.

The notes are written to be accurate (cross-checked with primary sources where possible), practical (focused on workflows, checklists, and decision points), and open (released under CC0 so they can be reused without attribution).
